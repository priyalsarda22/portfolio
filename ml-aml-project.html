AML Classification Project

Machine learning model for classifying Acute Myeloid Leukemia from flow cytometry data

Machine Learning • Medical Diagnostics • Python

Python scikit-learn Pandas NumPy Random Forest Flow Cytometry
← Back to Projects

Project Overview

Developed a machine learning model to classify Acute Myeloid Leukemia (AML) from flow cytometry data using Random Forest algorithms. This project demonstrated the application of supervised learning techniques to medical diagnostic data.

The goal of this project was to build a binary classifier that could accurately distinguish between normal and AML samples based on flow cytometry measurements from 8 different tubes with various cell surface markers (CD45, CD34, CD38, etc.). The dataset contained multiple fluorescence intensity measurements across different channels for each sample.

Technical Implementation

Data Processing: Implemented comprehensive feature engineering by extracting statistical features (mean, standard deviation, median, and quartiles) from each flow cytometry tube. Additionally calculated domain-specific ratios such as kappa/lambda light chain ratios and CD34/CD38 ratios, which are clinically relevant markers for hematological disorders.

Model Development: Utilized scikit-learn to build a Random Forest Classifier with optimized hyperparameters (1000 estimators, max depth of 5, and sqrt max features). Applied StandardScaler for feature normalization to ensure all measurements were on comparable scales. Implemented stratified train-test splitting to maintain class distribution given the imbalanced nature of the dataset (normal samples significantly outnumbered AML cases).

Model Evaluation: Achieved strong classification performance with training accuracy of approximately 0.95 and testing accuracy demonstrating good generalization. The model successfully handled the class imbalance challenge inherent in medical diagnostic datasets where disease cases are typically rare compared to normal samples.

Key Skills Demonstrated

  • Machine Learning Fundamentals: Supervised learning, binary classification, ensemble methods (Random Forest), model evaluation and validation
  • Data Processing: Feature extraction, feature engineering, data normalization, handling missing data and incomplete samples
  • Python Programming: Pandas for data manipulation, NumPy for numerical computations, scikit-learn for machine learning implementation
  • Domain Knowledge: Understanding of flow cytometry data, hematological markers, and their clinical significance in leukemia diagnosis
  • Model Optimization: Hyperparameter tuning, addressing class imbalance, preventing overfitting through proper validation strategies

Results and Impact

The final model successfully classified 180 samples with appropriate distribution predictions (161 normal, 19 AML), demonstrating robust performance on unseen data. This project showcased the practical application of machine learning in medical diagnostics, where accuracy and reliability are critical. The implementation emphasized best practices including proper data splitting, feature scaling, and validation techniques essential for deploying machine learning models in healthcare settings.
